### Installation

1.  **Clone the repo:**
    ```bash
    git clone [https://github.com/your-username/your-repo-name.git](https://github.com/your-username/your-repo-name.git)
    ```
2.  **Open in Android Studio:**
    `File > Open > myquizApp
3.  **Sync Gradle:**
    Wait for the project to download all necessary dependencies.
4.  **Run the app:**
    Select your device/emulator and click the **Run** (green play) button.
